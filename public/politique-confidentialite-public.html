Cette page présente de manière claire la façon dont les informations liées au site public SEAL peuvent être utilisées dans le cadre de l’expérience proposée.

Lecture simple

Le but de cette page est d’expliquer l’essentiel dans un langage accessible, sans complexité inutile.

Transparence

Nous privilégions une présentation claire de ce qui est affiché, enregistré localement ou transmis par le formulaire de contact.

Résumé

Les points essentiels à retenir.

Utilisation du site

Le site public SEAL permet principalement d’afficher des informations liées à la consultation d’un domaine et de proposer un formulaire de contact.

Certaines préférences, comme la langue ou le mode clair/sombre, peuvent être conservées localement dans le navigateur pour améliorer le confort d’utilisation.

Détails

Informations générales sur la confidentialité du site public.

1. Objet de la politique

La présente politique de confidentialité a pour but d’expliquer, de manière générale, comment les informations liées à l’utilisation du site public SEAL peuvent être traitées.

Elle s’applique aux pages publiques du site, notamment la page d’accueil, les pages d’information, la FAQ et le formulaire de contact.

2. Informations affichées ou traitées sur le site

Dans le cadre de l’expérience proposée, le site peut traiter des informations nécessaires à l’affichage des résultats demandés par l’utilisateur, par exemple lorsqu’un domaine est saisi dans l’interface prévue à cet effet.

Le site peut également utiliser les données volontairement renseignées dans le formulaire de contact, comme le nom, l’adresse e-mail, l’objet et le message.

3. Préférences enregistrées localement

Pour améliorer l’utilisation du site, certaines préférences peuvent être enregistrées localement dans le navigateur de l’utilisateur.

  • la langue sélectionnée dans l’interface ;
  • le mode clair ou sombre choisi ;
  • d’autres réglages purement liés à l’affichage ou au confort de navigation.

4. Utilisation du formulaire de contact

Lorsque l’utilisateur utilise le formulaire de contact, les informations qu’il fournit sont destinées à transmettre son message à l’équipe SEAL via l’adresse de contact prévue.

Ces informations sont utilisées dans le seul but de traiter la demande, d’y répondre, ou d’assurer le suivi de l’échange si cela est nécessaire.

5. Finalité générale

Les traitements réalisés dans le cadre du site public ont pour finalité principale d’assurer le bon fonctionnement de l’interface, de permettre l’affichage des contenus demandés, de conserver certaines préférences d’affichage, et de transmettre les messages envoyés via la page Contact.

6. Durée et conservation

Les préférences techniques conservées localement dans le navigateur restent sous le contrôle de l’utilisateur, qui peut les supprimer depuis les paramètres de son navigateur.

Les informations envoyées via le formulaire de contact peuvent être conservées le temps nécessaire au traitement et au suivi raisonnable de la demande.

7. Mise à jour de cette politique

Cette politique de confidentialité peut évoluer afin de refléter les changements apportés au site public, à son fonctionnement ou à sa présentation.

La version affichée sur cette page est celle qui s’applique au moment de la consultation.

Besoin d’une autre information ?
